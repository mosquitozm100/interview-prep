# Realtime

## 中文理解

实时系统关注连接管理、消息投递、顺序、在线状态和降级。不是所有消息都需要严格实时。

## English Interview Script

For realtime systems, I separate connection management from message persistence and delivery. I define ordering guarantees, retry behavior, and how clients recover after reconnecting.

## Key Topics

- WebSocket gateways
- Presence
- Fanout
- Ordering
- Offline delivery
- Backpressure

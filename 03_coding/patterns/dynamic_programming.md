# Dynamic Programming

## 中文理解

DP 的重点是状态定义。先问：子问题是什么？选择是什么？状态之间如何转移？初始化是什么？

## English Interview Script

I use dynamic programming when the problem has overlapping subproblems and an optimal substructure. I define the state, transition, base cases, and final answer before coding.

## Common Mistakes

- State missing necessary information
- Wrong iteration order
- Forgetting base cases
- Overcomplicating when greedy is enough

## Template / Mental Model

1. State definition
2. Transition
3. Initialization
4. Iteration order
5. Answer extraction
6. Space optimization if needed

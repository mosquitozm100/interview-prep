# Behaviour

Use this folder for behavioural interview stories, common questions, and resume pitch practice.

## Main Files

- `story_bank.md`: source of truth for stories
- `question_bank.md`: common behavioural questions
- `resume_pitch.md`: short positioning scripts
- `amazon_stories.md`: Amazon story drafts
- `uber_stories.md`: Uber / AI story drafts

## Rule

Every behavioural story should appear in `story_bank.md` before becoming a polished answer.
